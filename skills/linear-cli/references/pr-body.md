# pr-body

> Generate an Alavida PR body from Linear issues

## Usage

```
Usage:   linear pr-body --issues <issues>

Description:

  Generate an Alavida PR body from Linear issues

Options:

  -h, --help                 - Show this help.                                                    
  -w, --workspace  <slug>    - Target workspace (uses credentials)                                
  --issues         <issues>  - Comma-separated issue identifiers, e.g. ALA-123,ALA-124  (required)
```
